# Tutorials

Tutorials are guided, end-to-end journeys. Each one has a clear starting point, a clear ending
point, and something real to show for your work.

If you are new to Release Regent, start with the first tutorial.

## Available tutorials

### [Your first automated release](01-first-release.md)

Walk through every step from installation to a published GitHub release. You will install the
CLI, create a GitHub App, configure a repository, write conventional commits, and watch Release
Regent drive the entire release process.

**Time**: approximately 15 minutes.
**Outcome**: a working Release Regent installation and your first automated release.

---

### [Customise your changelog](02-customise-changelog.md)

Learn how to group commits differently, add emoji headers, skip certain commit types, and use
the full power of the Tera template engine to shape your changelog exactly the way your project
needs it.

**Time**: approximately 10 minutes.
**Outcome**: a changelog configuration tailored to your project's conventions.
